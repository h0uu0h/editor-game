# 腾讯新闻编辑部 — 养成合成小游戏策划设计方案

> **项目类型**：新闻资讯 × 合成养成小游戏
> **目标平台**：腾讯新闻 App 内嵌
> **设计者**：[候选人] × AI 协作
> **Demo 地址**：本地 React 项目 (React 18 + Vite)

---

## 一、项目背景与目标

### 1.1 业务目标

腾讯新闻面临资讯类产品共同的挑战：用户「用完即走」，日均访问时长和频次难以提升。需要一种**高粘性的游戏化机制**，让用户形成「浏览新闻 → 获得游戏资源 → 成长反馈 → 回来继续」的正向循环。

| 增长目标 | 游戏化手段 |
|---------|-----------|
| 提升 DAU / 回访率 | 生产者产出 → 离线收益 → 「回来收菜」心理 |
| 延长单次停留 | 合成上瘾 + 订单系统 + 闯关挑战 |
| 提升浏览深度 | 浏览新闻 = 免费获取线索（核心资源绑定） |
| 商业化空间 | 广告障碍格、加速卡、高级版面 IAP |

### 1.2 设计原则

1. **新闻调性（News Tone）**：游戏元素全部映射为新闻编辑室的真实概念——线索碎片、采访笔记、稿件、专栏、特刊，不能违和
2. **浏览即生产**：核心资源（线索碎片）通过浏览腾讯新闻免费获取，不设付费门槛，保持游戏与产品的强绑定
3. **易上手深策略**：前 30 秒学会拖拽合成，后续通过版面管理、多页系统、闯关变体规则制造策略深度
4. **长线养成闭环**：参考 Farm Merge Valley 的 5 货币循环，确保金币/声望永远有出口

## 二、核心玩法

### 2.1 玩法概述

玩家扮演腾讯新闻的「实习编辑」，在一张 7×7 的虚拟编辑桌上：
- **收集**线索碎片（Lv1），通过**拖拽合成**升级为采访笔记（Lv2）→ 选题方案（Lv3）→ 稿件（Lv4）→ 专栏 → 特刊 → 周刊 → 传奇刊物（Lv8）
- 完成**订单**（快报、深度调查、独家头条）赚取金币和声望
- 用金币**扩建版面**、解锁**新版面**（财经/体育/文化/国际）
- 挑战**采访任务**（闯关模式）获得额外奖励

**一句话定位**：「Farm Merge Valley，但是办报纸。」

### 2.2 合成规则（Merge 机制）

| 规则 | 说明 |
|------|------|
| **邻接合成** | 只有上下左右四方向相邻的同类型、同等级碎片才能合成（BFS 连通检测），不是全棋盘扫描 |
| **3 合 1** | 3 个相邻同类同级 → 合成 1 个 +1 级 |
| **5 合 2** | 5 个相邻同类同级 → 合成 2 个 +1 级（奖励机制，鼓励布局规划） |
| **Combo** | 2 秒内连续合成 → Combo 倍数递增 → 额外金币奖励 |
| **版面加成** | 专业版面（如财经版）内对应类型只需 2 合 1（降低门槛） |

### 2.3 碎片体系

**5 大新闻类型** × **8 级合成链** × **5 种素材形式** = 200 种碎片

#### 新闻类型

| 类型 | 色系 | 对应真实内容 |
|------|------|-------------|
| 🖥 科技 | 蓝色系 `#E6F1FB` | 科技频道内容 |
| 📈 财经 | 绿色系 `#EAF3DE` | 财经频道内容 |
| ⚽ 体育 | 橙色系 `#FAECE7` | 体育频道内容 |
| 🎨 文化 | 紫色系 `#EEEDFE` | 文化频道内容 |
| 🌍 国际 | 粉色系 `#FBEAF0` | 国际频道内容 |

#### 合成链（以科技为例）

```
📎 线索碎片 → 📝 采访笔记 → 💡 选题方案 → 💻 科技稿件* → 🖥️ 深度专栏* → 🤖 AI特刊* → 🏛️ 科技周刊* → 🚀 传奇刊物*
   Lv1          Lv2           Lv3          Lv4            Lv5            Lv6           Lv7           Lv8
```

> *Lv4+ 为「生产者」：自动产出 Lv1 线索到相邻空格，是放置 idle 养成的核心

#### 素材形式

| 形式 | 图标 | 作用 |
|------|------|------|
| 📋 事实 | fact | 基础素材，最常见 |
| 🗣️ 引语 | quote | 人物采访产出 |
| 📊 数据 | data | 数据报告必备 |
| 📷 图片 | photo | 图文报道必备 |
| 🔥 独家 | scoop | 稀有素材，卖价 ×1.5 |

## 三、三大系统架构

### 3.1 系统一：自由编辑模式（主界面 / 长线养成）

这是玩家大部分时间所在的模式。核心循环：

```
浏览腾讯新闻 → 获得线索碎片（免费）
         ↓
    在编辑桌上拖拽合成
         ↓
    Lv4+ 产出者自动生产 → 离线也产出（回来收菜）
         ↓
    完成订单 → 赚金币 + 声望
         ↓
    金币 → 扩建版面 / 解锁新版面
    声望 → 解锁新版面 / 编辑头衔
         ↓
    更大空间 → 更多合成 → 更高级产出者 → 更多产出 ...
```

#### 3.1.1 棋盘设计

- **7×7 网格**（49 格），初始可用约 31 格
- **锁定格**：底部 2 行 + 若干角落，需花金币逐步扩建
- **障碍格**：永久存在的广告/赞助位（📺广告位、🏷️推广、☕赞助商、📢公告栏），不可使用但符合报纸调性
- **扩建价格曲线**：80 → 200 → 500 → 1000 → 2000 → 3000 → 5000（永远花不完）

#### 3.1.2 订单系统

三个订单槽，消耗特定形式的碎片完成：

| 订单类型 | 需求 | 奖励 | 解锁条件 |
|---------|------|------|---------|
| Tier 1：突发快报 | 2 个素材 | 40-60 💰 + 10-20 🏆 | 初始 |
| Tier 2：数据报告 | 3 个素材 | 100-120 💰 + 45-50 🏆 | 发布 3 篇后 |
| Tier 3：独家头条 | 含独家素材 | 180-250 💰 + 90-150 🏆 | 发布 8 篇后 |

#### 3.1.3 突发新闻

随机事件（40-90 秒间隔），弹出「🔴 突发：重大科技突破！」等，自动往棋盘撒 3 个对应类型碎片 + 限时奖励。制造紧迫感和惊喜。

#### 3.1.4 卖出系统

点击碎片 → 弹出信息卡 → 确认卖出。用于清理棋盘、回收金币。

| 等级 | 卖价 | 独家加成 |
|------|------|---------|
| Lv1 | 5 💰 | 7 💰 |
| Lv4 | 100 💰 | 150 💰 |
| Lv8 | 2500 💰 | 3750 💰 |

### 3.2 系统二：多页报纸（Multi-Page Newspaper）

编辑部不只出一个版面。5 个独立版面，各有特色：

| 版面 | 图标 | 解锁门槛 | 特殊能力 |
|------|------|---------|---------|
| **头版 A1** | 📰 | 免费 | 综合版面，无加成 |
| **财经版 B1** | 📈 | 2000💰 + 200🏆 | 财经类碎片 **2合1** |
| **体育版 C1** | ⚽ | 5000💰 + 500🏆 | 体育类碎片 **2合1** |
| **文化版 D1** | 🎨 | 10000💰 + 1000🏆 | 文化类碎片 **2合1** |
| **国际版 E1** | 🌍 | 20000💰 + 2000🏆 | 国际类碎片 **2合1** |

**关键设计**：
- 每个版面是**独立的 7×7 棋盘**，有自己的障碍布局和锁定格
- 版面之间**共享**金币、声望、订单
- 每个版面内部扩建价格**独立计算**（各自 80→200→500→...）
- 底部 Tab 栏切换，生产者只在当前显示的版面运作
- **核心 Coin Sink**：5 个版面 × 7 级扩建 = 35 次购买 + 4 次版面解锁 = 几乎无限的金币去处

### 3.3 系统三：采访任务（Story / Challenge Mode）

闯关模式，独立于自由模式，给予玩家**目标感、成就感、规则新鲜感**。

#### 3.3.1 章节结构

| 章节 | 解锁条件 | 关卡数 | 主题 |
|------|---------|--------|------|
| **第一章：实习记者** | 免费 | 4 关 | 学习标准合成 |
| **第二章：外勤采访** | 8 ⭐ | 4 关 | 掌握特殊规则 |
| **第三章：深度调查** | 20 ⭐ | 3 关 | 复杂版面挑战 |

#### 3.3.2 六种合成规则变体

这是闯关模式的核心差异点——每关可以使用不同于自由模式的合成规则：

| 规则 | 图标 | 可合成方向 | 策略变化 |
|------|------|-----------|---------|
| ➕ 标准 | 上下左右 | 常规 |
| ✖️ 对角线 | 四个斜向 | 碎片要斜着排列 |
| ⏭️ 隔一格 | 隔一格的上下左右 | 要刻意留空隙 |
| ↔️ 同行限定 | 只有左右 | 横向思维 |
| ↕️ 同列限定 | 只有上下 | 纵向排列 |
| 🌟 八方合成 | 全 8 方向 | 最灵活，搭配高难度目标 |

#### 3.3.3 关卡限制类型

- ⏱ **时间限制**：倒计时，到 10 秒变红闪烁
- 👆 **步数限制**：每次拖拽消耗 1 步
- 两者可**叠加**（如 L1-4：35 秒 + 20 步双限制）

#### 3.3.4 三星评价

每关 3 颗星，独立条件：
- ⭐ 第一颗：完成基础目标
- ⭐⭐ 第二颗：时间/步数效率达标
- ⭐⭐⭐ 第三颗：额外挑战（合出更高等级等）

**重要设计**：目标达成后**不自动结算**——弹出绿色横幅「✅ 目标已达成！继续冲击三星，或提交结算」，玩家可以继续操作直到手动提交或限制耗尽。进度条右侧有**实时星星追踪**，已满足的条件即时亮金。

#### 3.3.5 可视化教学系统

每个关卡开始前弹出三步教学：
1. **规则可视化**：3×3（或 5×5）小网格，绿色 ✓ 标出可合成方向
2. **目标说明**：目标类型 + 合成链示意（Lv1 → Lv2 → Lv3）
3. **挑战条件**：限时/限步/障碍数 + 三星条件列表 + 通关奖励

教学结束点击「🚀 开始挑战」后才启动计时。

#### 3.3.6 奖励回流

通关奖励直接发放到自由模式的金币和声望，形成两个系统的正向循环。

#### 3.3.7 关卡列表（11 关）

| ID | 名称 | 规则 | 棋盘 | 目标 | 限制 | 难点 |
|----|------|------|------|------|------|------|
| L1-1 | 第一条线索 | 标准 | 5×5 | 2个Lv2 | 40秒 | 13碎片散布+干扰类型 |
| L1-2 | 素材分拣 | 标准 | 6×5 | 1个Lv3 | 50秒 | 16碎片+3障碍 |
| L1-3 | 截稿压力 | 标准 | 6×5 | 2个Lv3 | 18步 | 步数紧张+干扰碎片 |
| L1-4 | 主编考核 | 标准 | 6×6 | 1个Lv4 | 35秒+20步 | 双限制 |
| L2-1 | 斜线采访 | 对角线 | 6×6 | 3个Lv2 | 55秒 | 斜向排列 |
| L2-2 | 隔空取材 | 隔一格 | 7×5 | 1个Lv3 | 60秒 | 中间列全障碍 |
| L2-3 | 横版排列 | 同行 | 7×5 | 3个Lv2 | 16步 | 碎片分散各行 |
| L2-4 | 纵向突破 | 同列 | 6×7 | 1个Lv3 | 45秒 | 列中障碍断路 |
| L3-1 | 迷宫版面 | 标准 | 7×7 | 1个Lv4 | 75秒 | 10个障碍迷宫 |
| L3-2 | 对角迷局 | 对角线 | 7×7 | 1个Lv4 | 90秒+25步 | 双限制+8障碍 |
| L3-3 | 终极特刊 | 八方 | 7×7 | 1个Lv5 | 60秒 | Lv4→Lv5 |

## 四、经济系统设计

### 4.1 货币体系

参考 Farm Merge Valley 的 5 货币循环，本游戏设计了 4 层货币：

| 货币 | 图标 | 获取方式 | 消耗方式 |
|------|------|---------|---------|
| **线索碎片** | 📎 | 浏览新闻(免费)、AI采访、生产者产出 | 合成升级、完成订单 |
| **金币** | 💰 | 完成订单、卖出碎片、Combo奖励、闯关 | 扩建版面、解锁版面、刷新订单 |
| **声望** | 🏆 | 完成订单、闯关奖励 | 解锁版面、编辑头衔 |
| **星星** | ⭐ | 闯关三星 | 解锁后续章节 |

### 4.2 核心经济循环

```
         ┌──────────────────────────────────────────┐
         │                                          │
    浏览新闻 ──→ 线索碎片(免费) ──→ 合成 ──→ 高级碎片
         │                           │             │
         │                           ↓             ↓
         │                      完成订单 ──→ 金币 + 声望
         │                                    │      │
         │                    ┌───────────────┘      │
         │                    ↓                      ↓
         │              扩建版面 ──→ 更多空间     解锁版面
         │                    ↓                      │
         │              更多合成 ──→ 更强产出者       │
         │                    ↓                      │
         └──────── 回来收菜（离线产出） ←────────────┘
```

### 4.3 Coin Sink 分析（防通胀）

| Sink 类型 | 单价范围 | 深度（次数） | 说明 |
|-----------|---------|-------------|------|
| 版面内扩建 | 80-5000 | 每页 7 次 × 5 页 = 35 次 | 渐进式价格曲线 |
| 版面解锁 | 2000-20000 | 4 次 | 大额一次性消费 |
| 订单刷新 | 50-200 | 无限 | 花钱换更好订单 |
| 设施升级 | 500-5000 | 规划中 | 打字机、摄影棚等 |
| **总计** | — | — | 约 10 万+ 💰 才能全解锁 |

## 五、与腾讯新闻的绑定设计

### 5.1 浏览 = 线索（核心绑定）

这是本设计与通用游戏的最大差异点：**线索碎片不花钱，通过浏览真实新闻获得**。

| 浏览行为 | 获得 |
|---------|------|
| 阅读 1 篇文章 | 1-2 个线索碎片（匹配文章频道类型） |
| 阅读 3 分钟+ | 额外 1 个碎片 |
| 观看视频 | 1 个照片类碎片 |
| 分享/评论 | 1 个引语类碎片 |
| 每日首次打开 | 3 个随机碎片 |

**收件箱机制**：碎片先存入收件箱（上限 20），玩家回到游戏一键领取。制造「我有资源等着领」的回来动机。

### 5.2 内容引导

- **AI 采访按钮**：5 个新闻频道按钮，点击后随机生成该频道的 2-3 个碎片 + 一条真实新闻标题（从腾讯新闻 API 获取）
- **突发新闻事件**：可绑定真实热点，如世界杯期间体育类突发频率翻倍
- **订单标题**：使用真实新闻体裁（突发快报、数据报告、人物专访、深度调查）

### 5.3 广告位自然植入

棋盘上的**障碍格**不只是游戏机制，同时是原生广告位：
- 📺 广告位 → 品牌展示
- ☕ 赞助商 → 可点击跳转
- 🏟️ 赛事 → 体育频道入口
- 📊 行情屏 → 财经数据入口

广告收入与游戏体验**不冲突**——障碍格本身就是合成策略的一部分。

## 八、用户旅程

### 8.1 新手引导（前 5 分钟）

```
1. 进入编辑部 → 看到 3 个科技碎片已在桌上
2. 引导拖拽 → 合成 Lv2（📝 采访笔记）→ 粒子动效 + 成就感
```

### 9.3 核心算法

- **BFS 邻接连通检测**：`findConnectedSame()` — O(n) 时间复杂度，确保只有真正相邻的同类碎片才能合成
- **闯关自定义方向**：`findConnectedLevel()` — 接受自定义 `dirs` 数组，支持 6 种规则变体
- **生产者定时器**：`useRef` + `setInterval`，切换页面时清除/重建
- **随机空位**：`findRandomEmpty()` 全局随机 + `findEmptyNear()` 八邻域优先

## 十一、总结

「腾讯新闻编辑部」不是一个简单嫁接的小游戏，而是一套**深度绑定新闻产品的游戏化增长系统**：

1. **浏览即生产** — 把「看新闻」变成「收集线索」，浏览行为天然转化为游戏资源
2. **新闻调性全覆盖** — 从碎片命名到版面结构，每个元素都是新闻编辑室的映射
3. **三系统长线深度** — 自由模式养成 + 多页扩展 + 闯关变体，满足不同玩家类型
4. **经济闭环防通胀** — 参考 Farm Merge Valley 验证过的金币→土地→更多合成的正循环
5. **商业化自然融入** — 广告障碍格、激励视频、VIP 联动，不伤害游戏体验

> 核心一句话：**每一次浏览都是一次采访，每一次合成都是一篇报道。**
